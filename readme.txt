Persian date support for WordPress

== Description ==

This is a simple package to convert dates to Jalali (Persian or Shamsi) in posts, pages, comments.
 It also can convert English numbers to Persian numbers

List of some features:

* Shamsi (Jalali) date in Posts, comments, pages, archives, search, categories
* Shamsi (Jalali) date in Permalinks
* RTL and fixed tinymce editor
* Powerful and fast function for Persian numbers
* Cinverts English number to Persian
* Uses Javascript
* Low resources usage

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. To use change setting go to 'option' and choose 'Fars date setting'
